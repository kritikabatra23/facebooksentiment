Project Goal

The aim of Facebook Sentiment allows users had better understanding of their recent emotional changes and personality, at the same time, the software will give users the appropriate advice based on the analyzation results.
  • Core functions / features
  • The web application has the register function allows users register the application with their email.​
  • The web application has the login function allows users login with their emails and passwords.
  • The web application can connect to the Facebook website and let users log into their Facebook.​
  • The accounts in web application can be associated with Facebook accounts.
  • The web application can analyse the recent posts of users in Facebook by recording specific words.
  • The web application will give the result of personality analyzation to the user and show the appropriate video.
