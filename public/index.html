Project Description

Facebook sentiment is a web application that can connect to Facebook and get information from it. The website will gathering the posts of users in the Facebook and analyse the personality of the user. After the analysing, the user will be defined as one of the four main personalities. Each personality points to an educational video that is used for enlighten or encourage users. In addition, the web application can get the image of user when the user is interacting with the application. The application will also analyse the emotion of the user in real time to ensure the accuracy of the personality analyzation.
